# JZ_app

:一款简单的本地存储、云服务器存储的记账软件

博客地址：https://blog.csdn.net/Lemon_wzq/article/details/103628355


PS：后台数据服务使用的是bmob后端云，如果没用过，请先到 https://www.bmob.cn/?invite_code=@qcg 注册或自行百度，很好上手的，很适合个人开发者及小公司。
